# util_time_test.cpp - 时间格式转换测试

## 概述

此文件测试 Cycles 工具库中人类可读时间格式与秒数之间的相互转换函数。这些函数用于渲染进度显示、日志记录和用户界面中的时间信息展示。

## 测试用例

### TEST(time_human_readable_to_seconds, Empty)
- **功能**: 验证空字符串的解析和零秒的格式化。
- **验证数据**: `"" -> 0.0 秒`, `0.0 秒 -> "00:00.00"`

### TEST(time_human_readable_to_seconds, Fraction)
- **功能**: 验证仅包含小数部分的时间字符串解析。
- **验证数据**:
  - `".1" -> 0.1 秒`, `".10" -> 0.1 秒`
  - `0.1 秒 -> "00:00.10"`

### TEST(time_human_readable_to_seconds, Seconds)
- **功能**: 验证秒级时间字符串的解析和格式化。
- **验证数据**:
  - `"2.1" / "02.10" -> 2.1 秒`, `2.1 秒 -> "00:02.10"`
  - `"12.1" / "12.10" -> 12.1 秒`, `12.1 秒 -> "00:12.10"`

### TEST(time_human_readable_to_seconds, MinutesSeconds)
- **功能**: 验证分:秒格式时间字符串的解析和格式化。
- **验证数据**:
  - `"3:2.1" / "03:02.10" -> 182.1 秒`, `182.1 秒 -> "03:02.10"`
  - `"34:12.1" / "34:12.10" -> 2052.1 秒`, `2052.1 秒 -> "34:12.10"`

### TEST(time_human_readable_to_seconds, HoursMinutesSeconds)
- **功能**: 验证时:分:秒格式时间字符串的解析和格式化。
- **验证数据**:
  - `"4:3:2.1" / "04:03:02.10" -> 14582.1 秒`, `14582.1 秒 -> "04:03:02.10"`
  - `"56:34:12.1" / "56:34:12.10" -> 203652.1 秒`, `203652.1 秒 -> "56:34:12.10"`
