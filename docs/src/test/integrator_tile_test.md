# integrator_tile_test.cpp - 渲染图块大小计算测试

## 概述

此文件测试 Cycles 渲染器中图块（Tile）最佳尺寸计算函数 `tile_calculate_best_size`。该函数根据图像尺寸、采样数量和可用路径状态数量，计算最优的图块大小（宽度、高度和每像素采样数），以最大化 GPU/CPU 资源利用率。

## 测试用例

### TEST(tile_calculate_best_size, Basic)
- **功能**: 测试基本场景下的图块尺寸计算。
- **验证场景**:
  - **CPU 类场景**（路径状态数为 1）: 1920x1080 图像，无论采样数为 1 还是 100，图块大小均为 1x1x1（逐像素处理）
  - **充足路径状态**: 当路径状态数量足以容纳整幅图像的所有像素和采样时，图块大小覆盖整幅图像
    - 1920x1080, 1 采样, 路径状态 = 1920*1080 -> 图块 1920x1080x1
    - 1920x1080, 100 采样, 路径状态 = 1920*1080*100 -> 图块 1920x1080x100

### TEST(tile_calculate_best_size, Extreme)
- **功能**: 测试极端参数下的图块尺寸计算，确保不会溢出或产生不合理结果。
- **验证场景**:
  - 32x32 图像，262144 采样，131072 路径状态 -> 图块 1x1x512
  - 32x32 图像，1048576 采样，131072 路径状态 -> 图块 1x1x1024
  - 32x32 图像，10485760 采样，131072 路径状态 -> 图块 1x1x4096
  - 32x32 图像，超大采样数 (8192*8192*2)，1024 路径状态 -> 图块 1x1x1024
