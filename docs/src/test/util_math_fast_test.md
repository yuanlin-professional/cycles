# util_math_fast_test.cpp - 快速数学近似函数测试

## 概述

此文件测试 Cycles 工具库中快速数学近似函数（`fast_sinf` 和 `fast_cosf`）的精度。这些函数使用多项式近似代替标准库三角函数，在牺牲少量精度的前提下大幅提升计算速度，适用于渲染器中对性能要求高但允许微小误差的场景。

## 测试用例

### TEST(math, fast_sinf)
- **功能**: 验证 `fast_sinf` 函数在各种输入范围下的精度。
- **验证场景**:
  - **常规范围** (-2PI ~ 2PI): 在 1001 个均匀采样点上，验证 `fast_sinf` 与标准 `sinf` 的偏差不超过 1.9e-7
  - **精确 PI/2 值**: 验证 `fast_sinf(PI/2) ≈ 1.0` 和 `fast_sinf(-PI/2) ≈ -1.0`，误差 < 1e-9
  - **接近 PI/2 值**: 验证邻近 PI/2 的值精度满足 1e-9 容差
  - **大数值输入**: 验证 `fast_sinf` 在大输入值（15378.3 和 -78431.5）下的范围归约精度，容差为 2e-4

### TEST(math, fast_cosf)
- **功能**: 验证 `fast_cosf` 函数在各种输入范围下的精度。
- **验证场景**:
  - **常规范围** (-2PI ~ 2PI): 在 1001 个均匀采样点上，验证偏差不超过 4.5e-7
  - **精确 PI/2 值**: 验证 `fast_cosf(PI/2) ≈ 0.0`，误差 < 3e-7
  - **接近 PI/2 值**: 验证邻近 PI/2 的值精度
  - **大数值输入**: 验证大输入值下的范围归约精度，容差为 2e-4
