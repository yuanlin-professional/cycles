# util_math_float4_test.cpp - float4 向量数学运算测试

## 概述

此文件测试 Cycles 工具库中 `float4` 四维浮点向量类型的数学运算函数。与 `float3` 测试类似，使用 `Float4Test` 夹具类，为微架构能力检测预留扩展点。测试验证的是默认微架构优化（如 `util/optimization.h` 中定义）下的运算结果。

## 测试用例

### TEST_F(Float4Test, fmod)
- **功能**: 验证 `float4` 类型的取模（`fmod`）运算在多种输入条件下的正确性。
- **验证场景**:
  - **基本取模**: `fmod({1.2, 2.3, 3.4, 4.5}, 1.0)` 期望结果 `{0.2, 0.3, 0.4, 0.5}`
  - **非整数除数**: `fmod({1.2, 2.3, 3.4, 0.9}, 1.2)` 期望结果 `{0.0, 1.1, 1.0, 0.9}`
  - **除数远大于被除数**: `fmod({1.2, 2.3, 3.4, 0.0}, 1000000.0)` 期望结果不变
  - **大数取模**: `fmod({1999999.2, 2000000.3, 2000001.4, 0.0}, 1000000.0)` 验证大数精度
  - **负数处理**: `fmod({5.1, -5.1, 0.0, 0.0}, 3.0)` 期望 `{2.1, -2.1, 0.0, 0.0}`
  - **负除数**: `fmod({5.1, -5.1, 0.0, 0.0}, -3.0)` 验证负除数行为
- **精度容差**: 1e-6
