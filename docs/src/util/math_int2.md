# math_int2.h - 二维整数向量运算

## 概述

`math_int2.h` 为 `int2` 类型提供基本的算术和位运算符重载。int2 在 Cycles 中的使用场景较为有限，主要用于像素坐标对、2D 索引等简单整数二元组操作。该文件是整个数学库中最精简的之一。

## 核心函数

### 运算符重载
| 运算符 | 说明 |
|--------|------|
| `==` | 逐分量相等比较 |
| `+` / `+=` | 加法 |
| `-` | 减法 |
| `*` | 乘法 |
| `/` | 除法 |
| `>>` | 右移（标量） |
| `&` | 按位与 |
| `^` | 按位异或 |

## 依赖关系

- **内部头文件**: `util/defines.h`, `util/types_int2.h`
- **被引用**: 通过 `util/math.h` 间接被引用；在降噪器的 2D 索引操作中使用

## 实现细节 / 关键算法

所有运算均为纯标量实现，无 SIMD 优化。Metal 平台下通过 `__KERNEL_METAL__` 条件编译跳过这些运算符，因为 Metal Shading Language 内建支持 int2 的运算。

## 关联文件

- `util/types_int2.h` — int2 结构体定义
- `util/math_float2.h` — 对应的浮点 2D 向量运算
