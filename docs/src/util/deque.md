# deque.h - 双端队列类型别名封装

## 概述

`deque.h` 将标准库中的 `std::deque` 引入到 Cycles 命名空间中。这是一个极简的封装文件，仅包含一条 `using` 声明，使得项目中可以直接使用 `deque` 而无需 `std::` 前缀，保持代码风格一致性。

## 类与结构体

该文件不定义新的类或结构体，仅通过 `using` 声明引入标准库类型：

- `deque` -- `std::deque`（双端队列容器，支持两端高效插入/删除）

## 核心函数

无。该文件仅包含类型别名声明。

## 依赖关系

- **内部头文件**: 无
- **标准库**: `<deque>`
- **被引用**: 当前代码库中未发现直接引用该头文件的文件（可能用于外部集成或未来扩展）

## 关联文件

- `util/queue.h` -- 队列容器封装（`std::queue` 默认底层使用 `std::deque`）
- `util/list.h` -- 链表容器封装，同样是简单的类型别名
