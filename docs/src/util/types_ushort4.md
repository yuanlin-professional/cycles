# types_ushort4.h - 四维无符号短整数向量类型定义

## 概述

`types_ushort4.h` 定义了四维无符号短整数向量结构体 `ushort4`，包含 `x`、`y`、`z`、`w` 四个 `uint16_t` 分量。该类型为最精简的向量类型之一，仅包含结构体定义而不提供工厂函数或运算符重载。适用于需要 16 位无符号整数四元组的场景，如半精度数据打包、紧凑索引等。在 GPU 原生向量类型可用时使用平台内置定义。

## 类与结构体

### `struct ushort4`

仅在非原生向量类型环境下定义：

| 成员 | 类型 | 说明 |
|------|------|------|
| `x` | `uint16_t` | 第一分量（0-65535） |
| `y` | `uint16_t` | 第二分量（0-65535） |
| `z` | `uint16_t` | 第三分量（0-65535） |
| `w` | `uint16_t` | 第四分量（0-65535） |

总大小为 8 字节。

## 核心函数/运算符重载

本文件不定义任何函数或运算符重载。`ushort4` 的构造依赖聚合初始化（如 `ushort4 a = {x, y, z, w}`）。
