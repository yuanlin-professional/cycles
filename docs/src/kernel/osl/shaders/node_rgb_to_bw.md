# node_rgb_to_bw.osl - RGB 转黑白着色器

## 概述

该着色器将 RGB 颜色转换为灰度（黑白）值。使用符合 ITU-R BT.709 标准的亮度权重系数，与人眼对不同颜色的感知灵敏度相匹配。

## 着色器签名

```osl
shader node_rgb_to_bw(color Color = 0.0,
                      output float Val = 0.0)
```

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| Color | color | 0.0 | 输入 RGB 颜色 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| Val | float | 计算得到的灰度亮度值 |

## 实现逻辑

使用 ITU-R BT.709 标准的亮度系数进行加权求和：

```
Val = Color[0] * 0.2126 + Color[1] * 0.7152 + Color[2] * 0.0722
```

权重系数说明：
- **红色通道（R）**：0.2126 -- 人眼对红色感知中等
- **绿色通道（G）**：0.7152 -- 人眼对绿色最为敏感，权重最大
- **蓝色通道（B）**：0.0722 -- 人眼对蓝色感知最弱，权重最小

这三个系数之和为 1.0，确保纯白色 (1, 1, 1) 映射为 1.0，纯黑色 (0, 0, 0) 映射为 0.0。

## 对应 SVM 节点

对应 Cycles SVM 中的 `NODE_RGBTOBW` 节点。该节点在 Blender 节点编辑器中显示为"RGB 转 BW"（RGB to BW）节点。该转换在 Cycles 内部也被广泛用于需要从颜色获取标量亮度值的场景。
