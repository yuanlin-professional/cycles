# node_set_normal.osl - 设置法线着色器

## 概述

该着色器用于直接设置着色点的法线方向。它会覆盖全局变量 `N`（着色法线）并输出新的法线值。此着色器声明为 `surface` 类型，因为它需要修改表面着色上下文中的全局法线变量。属于 Cycles 渲染器开放着色语言（OSL）矢量工具节点。

## 着色器签名

```osl
surface node_set_normal(
    normal Direction = N,
    output normal Normal = N)
```

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| Direction | normal | N | 要设置的法线方向，默认为当前着色点的法线 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| Normal | normal | 设置后的法线输出 |

## 实现逻辑

1. 将全局着色法线 `N` 设置为输入的 `Direction` 值。
2. 将 `Direction` 值也赋给输出参数 `Normal`。

该着色器的关键作用是修改全局变量 `N`，使后续连接的着色器使用新的法线方向。这在需要自定义法线方向（如程序化法线生成）时非常有用。

## 对应 SVM 节点

对应 SVM 内核中的法线设置操作，类型为 `SHADER_NODE_NORMAL`。
