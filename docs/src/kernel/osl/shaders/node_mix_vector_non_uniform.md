# node_mix_vector_non_uniform.osl - 非均匀向量混合着色器

## 概述

该着色器对两个向量值使用向量因子进行逐分量独立的线性插值混合。与均匀向量混合不同，本着色器的混合因子也是向量类型，允许对每个分量使用不同的混合比例。属于 Cycles 渲染器开放着色语言（OSL）矢量工具节点。

## 着色器签名

```osl
shader node_mix_vector_non_uniform(
    int use_clamp = 0,
    vector Factor = 0.5,
    vector A = 0.0,
    vector B = 0.0,
    output vector Result = 0.0)
```

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| use_clamp | int | 0 | 是否将混合因子向量的每个分量钳制到 [0, 1] 范围 |
| Factor | vector | 0.5 | 混合因子向量，每个分量独立控制对应方向的插值比例 |
| A | vector | 0.0 | 第一个输入向量 |
| B | vector | 0.0 | 第二个输入向量 |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| Result | vector | 混合后的向量结果 |

## 实现逻辑

1. 根据 `use_clamp` 标志决定是否对因子向量的每个分量进行钳制。
2. 使用 `mix(A, B, t)` 内置函数执行逐分量的线性插值。由于 `t` 为向量类型，每个分量的混合比例可以不同。

例如，若 `Factor = (0.2, 0.8, 0.5)`，则：
- X 分量：`A.x * 0.8 + B.x * 0.2`
- Y 分量：`A.y * 0.2 + B.y * 0.8`
- Z 分量：`A.z * 0.5 + B.z * 0.5`

## 对应 SVM 节点

对应 SVM 内核中的 `NODE_MIX` 节点，类型为 `SHADER_NODE_MIX`（新版混合节点的向量非均匀模式）。
