# node_blackbody.osl - 黑体辐射着色器

## 概述

该着色器根据给定的色温（开尔文）计算黑体辐射的颜色。输出的颜色已按亮度归一化，仅保留色度信息，不包含强度信息。用于模拟物理上真实的热辐射颜色，常用于光源颜色设置。属于 Cycles 渲染器开放着色语言（OSL）转换工具节点。

## 着色器签名

```osl
shader node_blackbody(
    float Temperature = 1200.0,
    output color Color = 0.0)
```

## 输入参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| Temperature | float | 1200.0 | 色温值，单位为开尔文（K）。典型范围：800K（暗红）到 12000K（蓝白） |

## 输出参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| Color | color | 经亮度归一化后的黑体辐射颜色 |

## 实现逻辑

1. 调用开放着色语言（OSL）内置的 `blackbody()` 函数，根据输入色温计算对应的 RGB 颜色值。
2. 使用 `luminance()` 函数计算该颜色的感知亮度值 `l`。
3. 若亮度值非零，将颜色除以亮度值进行归一化，使输出颜色仅包含色度信息。
4. 这种归一化处理允许用户单独控制光源的强度，而着色器仅提供正确的色温色彩。

## 对应 SVM 节点

对应 SVM 内核中的 `NODE_BLACKBODY` 节点，类型为 `SHADER_NODE_BLACKBODY`。
