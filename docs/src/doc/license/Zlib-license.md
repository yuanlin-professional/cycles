# Zlib-license.txt - Zlib 许可证

## 概述

此文件包含 Zlib License（Zlib 许可证）的完整法律文本。Zlib 许可证是一种宽松的开源许可证，最初用于 zlib 压缩库。在 Cycles 项目中，部分从外部来源改编的代码使用此许可证，文件头部通过 SPDX 标识符 `SPDX-License-Identifier: Zlib` 进行标注。

## 关键条款

### 免责声明

- 软件按"原样"提供，不附带任何明示或暗示的保证。
- 作者不对因使用软件而产生的任何损害承担责任。

### 三项限制条件

1. **不得虚假声明来源**：不得歪曲软件的来源，不得声称自己编写了原始软件。如果在产品中使用此软件，在产品文档中进行致谢表示感谢，但这并非强制要求。

2. **标注修改版本**：修改后的源代码版本必须明确标记为修改版本，不得冒充原始软件。

3. **保留声明**：此声明不得从任何源代码分发中删除或修改。

## 在 Cycles 中的使用

Zlib 许可证与 Apache 2.0 许可证完全兼容，是 Cycles 项目中使用的辅助许可证之一。该许可证以其极简的条款著称，对再分发的限制非常少，主要关注防止对软件来源的虚假声明。在 Cycles 代码库中，使用此许可证的文件会通过 SPDX 标识符进行明确标注。
