# readme.txt - Cycles 许可证说明

## 概述

此文件是 Cycles 渲染器许可证目录的总览说明文件，概述了项目的许可证策略和代码来源标注方式。

## 关键条款

### 默认许可证

Cycles 源代码的默认许可证为 **Apache 2.0**。这意味着，除非源文件头部另有标注，所有 Cycles 源代码均受 Apache 2.0 许可证的约束。

### 兼容许可证代码

Cycles 项目中也包含从其他来源改编的代码，这些代码使用不同但兼容的开源许可证。这些代码通过以下方式进行标注：

- **SPDX 许可证标识符**：在源文件头部使用标准化的 SPDX 标识符声明适用的许可证类型
- **版权声明**：在源文件头部注明原始版权所有者信息

### 许可证索引

完整的许可证标识符列表及其对应的许可证文本文件，请参阅 `SPDX-license-identifiers.txt`。该文件提供了 Cycles 代码库中所有使用的许可证类型的快速对照索引。

## 在 Cycles 中的作用

此文件作为许可证目录的入口文档，帮助开发者和用户快速了解 Cycles 项目的许可证架构：以 Apache 2.0 为主，辅以 BSD-3-Clause、MIT 和 Zlib 等兼容许可证。
