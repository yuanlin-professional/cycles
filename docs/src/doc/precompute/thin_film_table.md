# thin_film_table.py - 薄膜干涉颜色匹配函数预计算表生成器

## 概述

此 Python 脚本用于预计算薄膜干涉（thin film interference）效果所需的颜色匹配函数（Color Matching Functions, CMF）查找表。脚本将 CIE 1931 2度标准观察者的 XYZ 颜色匹配函数从波长域重采样到频率域，然后通过快速傅里叶变换（FFT）生成频谱系数表，最终输出可直接嵌入 Cycles C++ 源代码的静态数组。

## 核心函数

### resample(func, freq)
- **功能**: 将颜色匹配函数从波长域重采样到频率域
- **参数**:
  - `func`: 三次样条插值函数（CubicSpline 对象），表示某个 CIE XYZ 通道的颜色匹配函数
  - `freq`: 频率数组（单位为频率空间坐标）
- **返回值**: 重采样后的函数值数组，已处理 NaN 值（替换为 0）
- **算法**: 通过 `1000 / freq` 将频率转换为波长（纳米），调用插值函数获取对应的 CMF 值，然后除以 `freq^2` 进行雅可比变换（Jacobian correction），将波长域的均匀采样转换为频率域的正确权重

## 依赖关系

| 依赖库 | 用途 |
|--------|------|
| `numpy` | 数值计算、数组操作、FFT 变换 |
| `scipy.interpolate.CubicSpline` | CIE XYZ 数据的三次样条插值 |

## 实现细节 / 关键算法

### 1. CIE 1931 XYZ 颜色匹配函数数据

脚本内嵌了完整的 CIE 1931 2度标准观察者 XYZ 颜色匹配函数数据（`CIEXYZ` 数组），覆盖波长范围 359nm 至 830nm（含终止行831nm处的零值），共 473 个数据点。每行包含四列：

- 第0列：波长（nm）
- 第1列：X 色匹配函数值
- 第2列：Y 色匹配函数值
- 第3列：Z 色匹配函数值

### 2. 三次样条插值

使用 `scipy.interpolate.CubicSpline` 对 X、Y、Z 三个通道分别创建三次样条插值函数（`cie_X`、`cie_Y`、`cie_Z`），以便在任意波长处获取平滑的 CMF 值。插值设置为不允许外推（`extrapolate=False`），超出数据范围的值返回 NaN。

### 3. 频率域重采样

关键参数：
- `endfreq = 60`：覆盖 0 到 60 微米的光程差范围
- `N = 1022`：采样点数，1022 个实数数据点经 FFT 后产生 512 个复数频谱值

频率数组通过 `np.linspace(0, pi*(N-1)/endfreq, N)` 生成等间距频率采样点，并通过 `np.clip` 将最小值限制为 `1e-40` 以避免除零错误。

`resample` 函数执行从波长到频率的域变换：
- 将频率 `freq` 转换为波长 `lambda = 1000 / freq`（单位为纳米）
- 通过插值获取对应波长处的 CMF 值
- 除以 `freq^2` 进行雅可比校正（因为 `d(lambda) = -1000/freq^2 * d(freq)`）

### 4. 快速傅里叶变换（FFT）

对重采样后的 X、Y、Z 三个通道分别执行实数 FFT（`np.fft.rfft`），将频率域信号变换到傅里叶空间。由于输入为实数序列，rfft 利用对称性仅返回正频率部分，输出 512 个复数值。

### 5. 输出格式

脚本将 FFT 结果格式化为 C 语言静态数组：

```c
static const float table_thin_film_cmf[512][6] = {
  { Re(X), Re(Y), Re(Z), Im(X), Im(Y), Im(Z) },
  ...
};
```

每行包含 6 个浮点数：X、Y、Z 三个通道的实部和虚部，使用科学计数法表示（7位有效数字）。

### 6. 物理背景

薄膜干涉是光线在薄膜（如肥皂泡、油膜）上下表面反射时产生的干涉现象。干涉颜色取决于薄膜厚度和入射光波长之间的关系。在渲染器中，通过预计算 CMF 的傅里叶变换系数，可以在运行时高效地计算任意薄膜厚度对应的干涉颜色，避免逐波长积分的高昂计算开销。频率域表示使得光程差（与薄膜厚度成正比）对颜色的影响可以通过简单的复数乘法和查表实现。
